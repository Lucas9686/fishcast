**Add `findNowIndex(pressureHistory)` helper:**
```javascript
function findNowIndex(pressureHistory) {
    var now = new Date();
    var currentHourStr = now.toISOString().slice(0, 13);
    return pressureHistory.findIndex(function(d) { return d.time.startsWith(currentHourStr); });
}
```
